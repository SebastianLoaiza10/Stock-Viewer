# Stock-Viewer
- Allows users to upload CSV files of historical stock data
- Automatically converts the data into candlesticks
- Lets users filter the candlestick range by date, and identify candlestick patterns of their choice
- Additionally, users can detect if a range of candlesticks is a peak or valley
- Utilizes the Fibonacci sequence to help evaluate when to buy or sell the stock based on confirmations
